    Règles du jeu

    1. Cliquez sur une carte pour la retourner.

    2. Cliquez sur une autre carte pour la retourner.

    3. Si les deux cartes sont identiques, elles restent retournées.

    4. Si les deux cartes sont différentes, elles se retournent automatiquement.

    5. Le jeu se termine lorsque toutes les cartes sont retournées.
